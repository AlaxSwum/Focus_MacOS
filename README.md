# Focus - macOS Menu Bar App

A beautiful and functional macOS menu bar app for managing your tasks, meetings, and schedule. Inspired by productivity apps like Rize and Raycast.

## Features

- **Menu Bar Integration** - Quick access from your menu bar
- **Today View** - See your upcoming and completed tasks
- **Todo List** - Manage personal todo items
- **Meetings** - Track your meetings and schedules
- **Task Reminders** - Get notified before tasks start
- **Full Calendar View** - See your schedule in a full window
- **Skip Tasks** - Skip tasks with optional reasons
- **Beautiful UI** - Modern macOS design with smooth animations

## Download & Installation

### Option 1: DMG (Recommended)
1. Download `Focus-macOS-v1.0.0.dmg`
2. Double-click to open the DMG
3. Drag `Focus-macOS.app` to the `Applications` folder
4. Open from Applications or Spotlight

### Option 2: ZIP
1. Download `Focus-macOS-v1.0.0.zip`
2. Extract the ZIP file
3. Move `Focus-macOS.app` to your Applications folder
4. Open from Applications or Spotlight

### First Launch (Important!)
When you first open the app, macOS may show a security warning because the app is not signed with an Apple Developer certificate.

To open the app:
1. Right-click (or Control-click) on `Focus-macOS.app`
2. Select "Open" from the menu
3. Click "Open" in the dialog that appears

Or go to **System Settings > Privacy & Security** and click "Open Anyway" next to the Focus app.

## Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon (M1/M2/M3) or Intel Mac

## How to Use

1. **Menu Bar**: Click the Focus icon in your menu bar to open the dropdown
2. **Tabs**: Switch between Today, Todo, and Meetings tabs
3. **Task Actions**: 
   - Click checkbox to mark complete
   - Double-click meetings to see details
4. **Full App**: Click "Open Full App" for the complete calendar view
5. **Notifications**: Get reminders 5 minutes before tasks

## Login

Use your Project Next account credentials to log in. The app syncs with your Project Next tasks and schedule.

## Building from Source

If you want to build from source:

```bash
# Clone the repository
git clone https://github.com/AlaxSwum/Focus_MacOS.git
cd Focus_MacOS

# Open in Xcode
open Focus.xcodeproj

# Build and run (Cmd + R)
```
